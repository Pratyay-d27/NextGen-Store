<h1 align="center">🚀 NextGen Store – Dynamic Animated Website</h1>

An ultra-modern, interactive, and visually rich **frontend web project** built using **HTML, CSS, and JavaScript**.  
This project demonstrates **dynamic rendering, smooth animations, 3D effects, and interactive UI elements** to create a futuristic shopping experience.

---

## 🌟 Features

### 🎯 Dynamic Product Rendering
- Products are generated dynamically using JavaScript
- Easy to scale and modify product data

### 🌀 Smooth Animations
- Scroll reveal animation using Intersection Observer
- Animated loading screen
- Glowing animated logo

### 🧊 3D Tilt Effect
- Cards rotate based on mouse movement
- Creates immersive and interactive UI

### 🪟 Interactive Modal
- Click on product cards to view details
- Smooth popup animation

### 🎨 Custom Cursor
- Stylish glowing cursor that follows user movement
- Enhances modern UI experience

### 🌌 Particle Background
- Canvas-based animated particles
- Lightweight and visually appealing background

### 🧭 Smooth Scrolling
- Seamless navigation experience

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & animations  
- **JavaScript (Vanilla JS)** – Logic & interactivity  
- **Canvas API** – Particle animation  
- **Intersection Observer API** – Scroll animations  

---

## 📂 Project Structure

```
NextGen-Store/
│── index.html     # Main structure
│── style.css      # Styling and animations
│── script.js      # Logic and dynamic rendering
```

---

## ⚙️ How It Works

### 1. Product Rendering
- Products are stored in an array
- JavaScript dynamically creates product cards

### 2. 3D Tilt Interaction
- Mouse movement controls card rotation
- Adds depth and interactivity

### 3. Scroll Reveal Animation
- Cards animate into view when scrolled

### 4. Modal System
- Displays product details on click

### 5. Particle Background
- Uses Canvas API for continuous animation
